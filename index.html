AzerothCore is a project based on a framework for MMO Applications that's built using C++ and MySQL for the DB content.

It's derived chronologically from Mangos -> TrinityCore -> Sunwell Core

Currently compatible with World of Warcraft 3.3.5a as an educational project


Our goals are very simple:

  • Offer a FULLY WORKING game experience!

    Before code optimization and other server features not related to game

  • Doing Software Development Right!

    We want to create a real development life cycle! Creating stable and fully working versions ( stable branches with support for only bug, security and crash fixes ) with all game contents playable. Followed by in developments versions where implementing new server features and optimizing code. These development branches will become stable themseves then.

  • Be Modular!

    Using submodules ,CMAKE dynamic inclusions and improving our script system. You will be able to use our framework to build your own MMO Project and easily extending it with your custom scripts.
