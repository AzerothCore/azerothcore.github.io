Azerothcore

View project on GitHub

AzerothCore is a project based on a framework for MMO Applications that's built using C++ using MySQL for the DB content.

It's derived chronologically from Mangos -> TrinityCore -> Sunwell Core

It's currently compatible with World of Warcraft 3.3.5a as an educational project


Our goals are very simple:

  • Offer a FULLY WORKING game experience!

    Before "code optimization" and other server features not related to game

  • Doing Software Development Right!

    We want to create a real development life cycle! Creating stable and fully working versions ( all game contents is playable), followed by "in developments" ( implementing features and optimizing code ) that will became stable themseves

  • Keeping compatibility with TrinityCore!

    We want to keep Database Structure and SOAP Interface ( WIP on new versions ) compatible with trinitycore. This to allow developers to create external tools and sql mods that can easily communicate/work for both applications

  • Be Modular! ( WIP )

    Using submodules ,CMAKE dynamic inclusions and improving our script system. You will be able to use our framework to build your own MMO Project and easily extending it with your custom scripts.
