366 // hooks are executed in following order, at specified event of spell:
367 // 1. BeforeCast - executed when spell preparation is finished (when cast bar becomes full) before cast is handled
368 // 2. OnCheckCast - allows to override result of CheckCast function
369 // 3a. OnObjectAreaTargetSelect - executed just before adding selected targets to final target list (for area targets)
370 // 3b. OnObjectTargetSelect - executed just before adding selected target to final target list (for single unit targets)
371 // 3c. OnDestinationTargetSelect - executed just before adding selected target to final target list (for destination targets)
372 // 4. OnCast - executed just before spell is launched (creates missile) or executed
373 // 5. AfterCast - executed after spell missile is launched and immediate spell actions are done
374 // 6. OnEffectLaunch - executed just before specified effect handler call - when spell missile is launched
375 // 7. OnEffectLaunchTarget - executed just before specified effect handler call - when spell missile is launched - called for each target from spell target map
376 // 8. OnEffectHit - executed just before specified effect handler call - when spell missile hits dest
377 // 9. BeforeHit - executed just before spell hits a target - called for each target from spell target map
378 // 10. OnEffectHitTarget - executed just before specified effect handler call - called for each target from spell target map
379 // 11. OnHit - executed just before spell deals damage and procs auras - when spell hits target - called for each target from spell target map
380 // 12. AfterHit - executed just after spell finishes all it's jobs for target - called for each target from spell target map
